Invoice Digital Generator - PWA

Ini adalah versi aplikasi/PWA dari invoice generator.

Cara upload ke GitHub Pages:
1. Upload semua file ini ke repository GitHub:
   - index.html
   - manifest.webmanifest
   - service-worker.js
   - background-invoice.png
   - folder icons
2. Buka Settings > Pages.
3. Pilih branch main dan folder /root.
4. Buka link GitHub Pages.

Cara install di HP Android:
1. Buka website GitHub Pages dari Chrome.
2. Tunggu sampai tombol "Install Aplikasi" muncul.
3. Tekan tombol tersebut.
4. Aplikasi akan masuk ke layar utama HP.

Jika tombol tidak muncul, tekan menu titik tiga di Chrome lalu pilih "Tambahkan ke layar utama" atau "Install aplikasi".

Mengganti background:
Ganti file background-invoice.png dengan gambar invoice baru, gunakan nama file yang sama.
